# BioMBTI - 生物专业方向测试

一个帮助生物专业学生探索适合自己的研究方向和职业道路的性格测试工具。

## 测试维度

测试基于9个维度评估你的倾向：

1. **抽象 / 具体** - 对抽象概念 vs 具体现象的理解偏好
2. **理论 / 应用** - 基础理论研究 vs 实际应用
3. **结构 / 探索** - 按部就班 vs 自由探索
4. **独立 / 协作** - 独自工作 vs 团队合作
5. **延迟 / 即时** - 长期投入 vs 即时反馈
6. **声望 / 收入** - 学术声望 vs 经济回报
7. **精确 / 容错** - 对精确度的要求
8. **稳定 / 冒险** - 稳定偏好 vs 冒险精神
9. **干实验 / 湿实验** - 计算分析 vs 动手实验

## 评分算法

1. 用户回答36道题（每题1-10分）
2. 根据题目方向（正向/反向）计算各维度得分
3. 使用欧氏距离计算用户得分与各专业方向标准分的距离
4. 返回距离最近的5个专业方向

## 使用
